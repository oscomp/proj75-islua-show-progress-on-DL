# proj75-islua-show-progress-on-DL

为iSulad 镜像下载过程添加进度条显示

### *描述* 

当前执行isula pull命令下载镜像时没有任何提示信息，无法知道当前的下载的进度，需要增加下载进度的显示，下载进度显示的打印信息可以参考docker下载镜像时的打印信息。
